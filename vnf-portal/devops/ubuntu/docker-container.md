# RUN UBUNTU IMAGE
```bash
docker run --name vnf-ubuntu --network vnf-network -p 80:8080 -p 443:8443 -p 22:22 -itd ubuntu:latest
```

# ENTER INTO UBUNTU CONTAINER
```bash
docker exec -it vnf-ubuntu bash
```
